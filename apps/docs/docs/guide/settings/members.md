# 成员与权限

路径：系统设置 → **成员与权限** `/settings/members`

> 状态：建设中（Phase 1 交付）

## 角色矩阵

| 角色 | 权限 |
|---|---|
| Owner | 全部权限，包括删除项目 |
| Admin | 配置修改、成员邀请、告警规则 |
| Member | 只读查看 Dashboard |

## 邀请流程

1. 项目 Owner / Admin 点击「邀请成员」
2. 填入邮箱 + 选择角色
3. 系统发送邀请链接（有效期 7 天）
4. 受邀者注册后自动加入该项目

## 多项目继承

成员以**项目**为单位授权，不同项目可持有不同角色。组织级租户支持在后续 Phase 扩展统一角色管理。

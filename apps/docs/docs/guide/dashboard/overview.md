# 数据总览

路径：Dashboard → **数据总览** `/dashboard/overview`

> 状态：建设中（Phase 6 交付）

## 能力规划

跨模块的健康度汇总视图，一屏回答三个问题：

1. **当下有没有事故？** —— 异常 / 5xx / CWV 突增聚合为一张健康度卡
2. **流量分布健康吗？** —— 实时 QPS、UV、关键页面 PV 叠加趋势
3. **长尾有没有积压？** —— Top 未解决 Issue / AI 待审 PR 待办数

## 在此之前

各维度数据已独立可用，建议先按场景去对应大盘：

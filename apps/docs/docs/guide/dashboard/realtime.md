# 实时监控

路径：Dashboard → **实时监控** `/dashboard/realtime`

> 状态：建设中

## 能力规划

最近 5 分钟的事件实时滚动面板，面向线上发布期 / 故障排查期：

- **事件流**：异常 / 5xx / 高延迟请求按时间倒序流式刷新
- **实时 QPS / 错误率**：秒级精度曲线
- **在线会话数**：当前活跃 session 近似统计
- **发布前后对比**：基于 `release` 标签的 A/B 分组

## 在此之前

需要看"最近 N 分钟"的数据，可在对应大盘顶栏切换 `range=1h` 时间窗口，接口层面已支持秒级精度的 `windowHours` 参数（≥ 1h）。
